# Nexus Development Prompt — Medium & Lower Priority Improvements

Continue development of the Nexus platform using the existing architecture.

Tech stack requirements:

* React frontend
* Node.js backend
* PostgreSQL database
* Maintain existing architecture and completed functionality
* Use migrations for any schema changes
* Keep the codebase portable (future AWS deployment)

This phase focuses on **usability improvements, workflow acceleration, and system polish**.

---

# Features to Implement

## 1. Draft New Email

Add support for composing brand-new emails.

### Requirements

* Compose new email from anywhere in the system
* Allow selecting sender mailbox:

  * shared mailbox
  * personal mailbox (only if owned by the current user)
* Support:

  * To / CC / BCC
  * Subject
  * Rich text body
  * Attachments
* Save draft capability
* Send email
* Ensure sent messages:

  * appear in the correct thread
  * sync properly with mailbox history

### UI

Add **"New Email"** to the global create menu.
